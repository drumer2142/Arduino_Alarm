# Arduino_Alarm

The project Arduino Alarm is a web srapper only for use inside greece.
It scrapes strikes data and also namedays.
It uses MongoDB to store the data.
The data stored are:

- Today's day name
- Time (depends on the cronjob)
- Year 
- Month
- Today's day number
- Today's strikes
- Tommorow strikes
- Today's name days
- Tommorow name days
